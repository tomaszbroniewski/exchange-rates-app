# Currency Exchange App

## Stack:
- .NET **8** (ASP.NET Core)
- Angular **18**
- Entity Framework Core **InMemory**

## Database:
- In-memory database (no external setup required)
- Default user:  
  - **Login:** `user`  
  - **Password:** `qwerty`

## Setup:
1. Clone the repo
2. Backend: Directory `ExchangeRatesAppApi`
3. Frontend: Directory `ExchangeRatesApp`
4. Open `http://localhost:4200`
